# GLUE（General Language Understanding Evaluation）

是一个英文语言理解基准测试平台，包含多个 NLP 任务，用于评估预训练语言模型的理解能力。就像 AI 模型的"考试系统"，包含多道"
题目"（子任务）。

## GLUE 包含 9 个子任务：

- CoLA（语法判断）
- SST-2（情感分析）
- MRPC（语义相似度）
- STS-B（语义相似度评分）
- QQP（问题相似度）
- MNLI（多类型推理）
- QNLI（问答推理）
- RTE（文本推理）
- WNLI（代词指代）

---                                                                                                                                                                                
## MRPC（Microsoft Research Paraphrase Corpus）

是 GLUE 中的一个子任务，微软研究院发布的句子对数据集。

任务类型：二分类 — 判断两个句子是否表达相同含义（是否是 paraphrase）

数据示例：
```bash
┌─────────────────────────────┬─────────────────────────────┬──────────┐                                                                                                           
│ sentence1                   │ sentence2                   │ label    │                                                                                                          
├─────────────────────────────┼─────────────────────────────┼──────────┤                                                                                                           
│ "The stock fell 10% today." │ "Shares dropped 10% today." │ 1 (相同)  │                                                                                           
├─────────────────────────────┼─────────────────────────────┼──────────┤
│ "I love pizza."             │ "I hate pizza."             │ 0 (不同)  │                                                                                                           
└─────────────────────────────┴─────────────────────────────┴──────────┘
```

数据规模：

- 训练集：约 3,668 条
- 验证集：约 408 条
- 测试集：约 1,725 条


---                                                                                                                                                                                
### 总结

- GLUE = 整个考试平台（多个任务的集合）
- MRPC = 其中一道题目（判断句子是否同义）

代码中 load_dataset("glue", "mrpc") 就是从 GLUE 基准中加载 MRPC 这个具体任务的数据。
